# Chapter 13. Operators

This chapter contains examples from the book **"The First Programmer's Book"**.  
We explore different types of operators in Python: arithmetic, assignment, comparison, logical, identity, membership, bitwise, and operator precedence.

---

## Folder Structure

- [**`13_code_samples.py`**](./13_code_samples.py) — all code snippets from the book's subsections compiled into one file.  
- [**`13_comparison_operators_program.py`**](./13_comparison_operators_program.py) — a program to check your favorite programming language using comparison operators.  
- [**`13_a_f_grading_program.py`**](./13_a_f_grading_program.py) — a program for grading on an A–F scale.  

---

## Contents of Examples

- **13.2 Assignment Operators** → basic assignment `=` and compound assignment operators `+=`, `-=`, `*=`, `/=`, `**=`, and more.  
- **13.3 Comparison Operators** → examples of comparisons `==`, `!=`, `<`, `>`, `<=`, `>=`.  
- **13.4 Program with Comparison Operators** → [**`13_comparison_operators_program.py`**](./13_comparison_operators_program.py).  
- **13.5 Logical Operators** → `and`, `or`, `not`, examples with numbers and conditions.  
- **13.6 Program for A–F Grading** → [**`13_a_f_grading_program.py`**](./13_a_f_grading_program.py).  
- **13.7 Identity Operators** → examples with `is`, `is not`, including checks for `None`.  
- **13.8 Membership Operators** → examples with `in`, `not in` for strings and lists.  
- **13.10 Operator Precedence** → a table of execution order and examples with parentheses.  
- **13.12 Self-Study** → code examples from the self-study section.  

---

## How to Run Examples

This chapter includes both short examples and complete programs.  
To run them, use Python in the terminal (or an online interpreter).  

⚠️ Not all examples produce output on screen. To see results, add `print()` statements.

Running an individual program:

```bash
python3 13_a_f_grading_program.py
```

```bash
python3 13_comparison_operators_program.py
```
