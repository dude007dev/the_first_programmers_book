# Розділ 13. Оператори

Цей розділ містить приклади з книги **"Перша книга програміста"**.  
Ми розглядаємо різні типи операторів у Python: арифметичні, присвоєння, порівняння, логічні, ідентичності, приналежності, бітові та порядок їх виконання.

---

## Структура папки

- **`13_code_samples.py`** — усі уривки коду з підрозділів книги, зібрані в одному файлі.  
  ⚠️ Деякі приклади призначені лише для демонстрації (наприклад, виведення результатів операторів), і їхня користь у навчанні важливіша, ніж самостійний запуск.  
- **Окремі програми** — повні приклади, які можна запускати:
  - `13_comparison_operators_program.py` — перевірка улюбленої мови програмування за допомогою операторів порівняння.  
  - `13_a_f_grading_program.py` — програма для оцінювання за шкалою A–F.  

---

## Зміст прикладів

- **13.2 Оператори присвоєння** → базове присвоєння `=` та скорочені оператори `+=`, `-=`, `*=`, `/=`, `**=` тощо.  
- **13.3 Оператори порівняння** → приклади перевірок `==`, `!=`, `<`, `>`, `<=`, `>=`.  
- **13.4 Програма з оператором порівняння** → [**`comparison_program.py`**](./13_comparison_operators_program.py).  
- **13.5 Логічні оператори** → `and`, `or`, `not`, приклади з числами й умовами.  
- **13.6 Програма для оцінювання за шкалою A–F** → [**`grading_program.py`**](./13_a_f_grading_program.py).  
- **13.7 Оператори ідентичності** → приклади з `is`, `is not`, зокрема перевірка на `None`.  
- **13.8 Оператори приналежності** → приклади з `in`, `not in` для рядків і списків.  
- **13.10 Оператори пріоритету** → таблиця порядку виконання та приклади з дужками.  
- **13.12 Самостійна робота** → приклади коду з самостійної роботи.  

---

## Як запускати приклади

1. Запустіть окрему програму, наприклад:
   ```bash
   python3 13_a_f_grading_program.py
   ```
