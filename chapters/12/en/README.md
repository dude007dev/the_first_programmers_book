# Chapter 12. Conversion of Basic Data Types

This section contains examples from the book **"The First Programmer's Book"**.  
We examine converting one data type to another using Python's built-in functions.

---

## Folder Structure

- [**`12_code_samples.py`**](./12_code_samples.py) — all code snippets from the book's subsections, collected in one file.  
  ⚠️ In this section, the examples are code fragments, not complete programs.

---

## Content of Examples

- **12. Conversion of Basic Data Types** → examples of conversion between `int`, `float`, `str`, `bool`, as well as between collections (`set`, `list`, `tuple`).
- **12.1 Conversion to Numeric Data Type** → program for calculating deposit with user input and value conversion.

---

## How to Run the Examples

This section contains only short examples, without ready-made programs.  
To check them:  

1. Copy the required code block.  
2. Paste it into a file (for example, `example.py`) or an online interpreter.  
3. Run the file in the terminal:  
    ```bash
    python3 example.py
    ```
    or run the program in an online interpreter.

⚠️ Not all examples are shown with output to the screen. If information is not displayed automatically, use the `print()` function.
