# Chapter 21. List Data Type

This chapter contains examples from the book **"The First Programmer's Book"**.  
We cover the list data type, its methods, and usage examples in detail.

---

## Folder Structure

- [**`21_code_samples.py`**](./21_code_samples.py) — all code snippets from the book sections collected in one file.
  ⚠️ Some examples intentionally demonstrate errors (like `IndexError`, `ValueError`) to explain typical situations.

---

## Contents of Examples

- **21. List Data Type** → creating lists, accessing elements by index, empty lists, zero-based indexing rule.
- **21.1 Iterating Through Lists** → examples of `while` and `for` loops for iterating through list elements.
- **21.2 enumerate() Function for Tracking Current Index** → examples of using `enumerate()` and unpacking values (index + element).
- **21.3 Operations on Lists. List Data Type Methods** → practical examples of `append()`, `remove()`, `insert()`, `pop()`, `sort()`, `reverse()`, `copy()`, `extend()`, `count()`, `index()` and more.
- **21.4 Application Examples** → working examples of programs using lists: "The Magic 8 Ball", sorting friends, reversing a word, changing element order.
- **21.5 List Comprehension** → examples of creating new lists in one line, filtering data in expressions.

---

## How to Run Examples

This chapter contains only short examples, without complete programs.  
To test them:  

1. Copy the required code block.  
2. Paste it into a file (for example, `example.py`) or in an online interpreter.  
3. Run the file in the terminal:  
    ```bash
    python3 example.py
    ```
    or run the program in an online interpreter.
