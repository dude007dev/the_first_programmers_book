# Chapter 7. Variable

This chapter contains examples from the book **"The First Programmer's Book"**.  
We introduce variables in Python: what they are, how they look, common types, how to create them, and how to use them.

---

## Folder structure

- [**`7_code_samples.py`**](./7_code_samples.py) — all code snippets from the chapter collected in a single file.  
  ⚠️ In this chapter the examples are code fragments, not full programs.

---

## Contents of examples

- **7.1 What is a variable** → creating a variable.
- **7.2 How to choose a variable name** → examples of good and bad names.  
- **7.3 Variable types** → examples of basic types (int, float, str, bool, None).  
- **7.4 Reassigning a variable** → examples of changing a variable's type.  
- **7.5 Variable size** → determining a variable's size in memory using `sys.getsizeof()`.

---

## How to run the examples

This chapter contains only short examples, not full programs.  
To try them out:  

1. Copy the code block you need.  
2. Paste it into a file (for example, `example.py`) or into an online interpreter.  
3. Run the file in the terminal:  
    ```bash
    python3 example.py
    ```
    or run it in an online interpreter.

⚠️ Not all examples print output. If an example does not show output automatically, use the `print()` function.
