## How to run the examples

This chapter includes short snippets and a complete program.

To test examples:  

1. Copy the code block you want.  
2. Paste it into a file (for example, `example.py`) or into an online interpreter.  
3. Run the file in a terminal:  
    ```bash
    python3 example.py
    ```
    or run it in an online interpreter.


To run the standalone program (from the chapter directory, e.g., from `.../chapters/04/en`):
```bash
python3 4_the_first_program.py
```
