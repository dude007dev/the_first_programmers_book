# Chapter 6. Comments

This chapter contains examples from the book **"The First Programmer's Book"**.  
We cover comments in Python: what they are, how they look, and when to use them.

---

## Folder structure

- [**`6_code_samples.py`**](./6_code_samples.py) — all code snippets from the chapter collected in one file.  
  ⚠️ In this chapter the examples are code snippets rather than complete programs.

---

## Examples contents

- **6.2 Single-line comments (inline)** → examples of comments on the same line as code.
- **6.3 Multi-line comments (block)** → examples of multi-line comments in Python.

---

## How to run the examples

This chapter contains only short snippets, not complete programs.
To test them:

1. Copy the code block you need.
2. Paste it into a file (for example, `example.py`) or into an online interpreter.
3. Run the file in a terminal:
    ```bash
    python3 example.py
    ```
    or run it in an online interpreter.
