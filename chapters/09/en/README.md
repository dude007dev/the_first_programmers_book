# Chapter 9. The Magic 8 Ball

This chapter contains examples from the book **"The First Programmer's Book"**.  
We write a program that illustrates the operation of the Magic 8 Ball device.

---

## Folder Structure

- [**`9_code_samples.py`**](./9_code_samples.py) — all code snippets from the book's subsections, collected in one file.  
- [**`9_the_magic_8_ball_program.py`**](./9_the_magic_8_ball_program.py) — the complete Magic 8 Ball program.  

---

## Content of Examples

- **9.1 Creating a list of answers** → working with a list.  
- **9.2 Creating a question** → user input of the question.  
- **9.3 Selecting an element from the list of answers** → how to get an element by index.  
- **9.4 Intermediate result** → code of a simplified version of the program.  
- **9.5 Selecting a random answer** → example of using `random.randint()`.

---

## How to Run the Examples

This chapter presents both short examples and a ready-made program.

To test the examples:  

1. Copy the required code block.  
2. Paste it into a file (for example, `example.py`) or an online interpreter.  
3. Run the file in the terminal:  
    ```bash
    python3 example.py
    ```
    or run the program in an online interpreter.

⚠️ Not all examples are shown with output to the screen. If information is not displayed automatically, use the `print()` function.

Running a separate program (from the appropriate directory, for example from `...chapters/09/en`):
```bash
python3 9_the_magic_8_ball_program.py
```
