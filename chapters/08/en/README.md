# Chapter 8. Data Types

This section contains examples from the book **"The First Programmer's Book"**.  
We get acquainted with data types in Python: numbers, boolean type, None, string type (str), bytes,
list, tuple, range, set, frozen set (frozenset), dictionary.

---

## Folder Structure

- [**`8_code_samples.py`**](./8_code_samples.py) — all code snippets from the book's subsections, collected in one file.  
  ⚠️ In this section, the examples are code fragments, not complete programs.

---

## Content of Examples

- **8.1 Data Types** → overview of different data types.  
- **8.2 Special Data Types** → examples of how to distinguish a special data type from a simple one.  
- **8.3 Mutable and Immutable Data Types** → examples of mutable and immutable data types.  

---

## How to Run the Examples

This section contains only short examples, without ready-made programs.  
To check them:  

1. Copy the required code block.  
2. Paste it into a file (for example, `example.py`) or an online interpreter.  
3. Run the file in the terminal:  
    ```bash
    python3 example.py
    ```
    or run the program in an online interpreter.

⚠️ Not all examples are shown with output to the screen. If information is not displayed automatically, use the `print()` function.
